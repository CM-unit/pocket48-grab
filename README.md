# pocket48-grab
口袋48直播&amp;录播web地址&amp;视频源获取

# 说明
1、此版本目前为alpha测试版本，仅实现了相关功能，希望能抛砖引玉，有更多的人一起完善它

2、直播源推荐使用VLC media player播放，该播放器可播放m3u8文件

3、经测试，部分弹幕地址出问题是官方的锅，官方的弹幕地址404。

时间稍久的录播才有lrc弹幕，可以将其转换为标准字幕文件配合播放器使用。
